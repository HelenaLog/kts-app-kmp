# Smartbot Pro

---

## 🎯 О проекте

SmartBot Pro - кроссплатформенное мобильное приложение для конструктора чат-ботов и ИИ-ассистентов от компании KTS. Оно позволяет операторам в реальном времени общаться с пользователями через Telegram, VK, WhatsApp, Jivo и другие каналы, управлять сценариями и ботами прямо со смартфона.

---
## Функционал

### 🧭 Навигация
- **Bottom Navigation** — Список диалогов / Профиль
- **TopBar** — выбор активного пространства (кабинет → проект)
### 1️⃣ Онбординг
- Карусель с описанием возможностей SmartBot Pro
- После просмотра — переход к авторизации
### 2️⃣ Авторизация
- Вход по логину и паролю
- Авторизация на основе **cookie-сессии** (Set-Cookie → хранилище → подстановка в запросы)
- Автоматический логаут при протухании куки
- Интеграция **Yandex SmartCaptcha** через WebView
### 3️⃣ Выбор пространства
- Список кабинетов и проектов в TopBar
- Автоматический выбор пространства по умолчанию
- Переключение между пространствами
### 4️⃣ Список диалогов
- Вкладки: **Все обращения** / **Ждут ответа**
- Для каждого диалога: аватарка, имя, превью последнего сообщения, иконка канала
- Pull-to-refresh, пагинация
- Real-time обновления через **WebSocket**
- Автоматическая отметка диалога как прочитанного при открытом чате
### 5️⃣ Поиск и фильтрация
- Поиск по имени пользователя
- Фильтрация по каналу, типу канала, спискам пользователей
### 6️⃣ Чат
- Список сообщений: пользовательские / бот / системные
- Отправка текстовых сообщений и файлов
- Превью изображений
- Скролл к последнему сообщению
- Real-time получение новых сообщений
### 7️⃣ Управление ботом
- Остановить / продолжить работу бота
- Запуск бота с определённого сценария и блока
### 8️⃣ Уведомления
- In-app баннеры через **Firebase Remote Config**
- Два типа: информационный и ошибка
- Действие по нажатию (открытие ссылки)
- Закрытие баннера сохраняется между сессиями
### 9️⃣ Профиль
- Краткая информация о профиле
- Кнопка логаута
### ⚙️ Общее
- Только портретная ориентация
- Адаптивность под разные размеры экранов
- Поддержка `minSdk = 23`, `targetSdk = 35`

## 🏗 Архитектура

**Clean Architecture** с разделением на слои внутри каждой фичи:

```
feature/
├── data/
│   ├── remote/        # Ktor API, DTO
│   ├── local/         # Room entities, DAO
│   ├── mapper/        # DTO → Domain
│   └── repository/    # реализация репозитория
├── domain/
│   ├── model/         # доменные модели
│   ├── repository/    # интерфейс репозитория
│   └── usecase/       # бизнес-логика
├── presentation/
│   ├── model/         # UI модели
│   ├── mapper/        # Domain → UI
│   ├── components/    # Composable компоненты
│   └── *ViewModel     # ViewModel + UiState + UiEvent
└── di/                # Koin модуль фичи
```

## 🛠 Технологии

### Multiplatform
| | |
|---|---|
| **Kotlin Multiplatform** | Общий `domain`, `data`, `presentation` слои |
| **Compose Multiplatform** | Общий UI для Android и iOS |

### DI & Async
| | |
|---|---|
| **Koin** | Dependency Injection |
| **Kotlin Coroutines + Flow** | Асинхронность и реактивность |

### Network
| | |
|---|---|
| **Ktor 3** | HTTP клиент, multipart upload, WebSocket |
| **Kotlinx Serialization** | JSON |
| **Centrifuge** | WebSocket протокол (real-time) |

### Storage
| | |
|---|---|
| **Room** | Локальная БД (кэш диалогов, кабинетов, проектов) |
| **DataStore** | Сессия, настройки, онбординг |

### UI & Utils
| | |
|---|---|
| **Coil** | Загрузка изображений |
| **FileKit** | Выбор файлов |
| **Napier** | Логирование |
| **Yandex SmartCaptcha** | WebView капча на экране входа |
 
---

## 📡 Real-time

WebSocket на базе протокола **Centrifuge**:
- Получение новых сообщений в чате
- Обновление списка диалогов без перезагрузки
- Обновление счётчика непрочитанных
- Автоматическое переподключение с экспоненциальным backoff
